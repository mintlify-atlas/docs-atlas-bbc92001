## What is Notification Center?

Notification Center is a Laravel Nova package that gives your application a production-ready multi-channel notification system. Define delivery strategies once — the package handles scheduling, channel escalation, retries, and tracking automatically.

## Key features

<AccordionGroup>
  <Accordion title="Automatic channel escalation">
    Configure an ordered list of channels per strategy. If a channel times out, the package automatically escalates to the next channel in the list — ensuring delivery even when one channel fails.
  </Accordion>
  <Accordion title="Smart retries with escalating delays">
    Define `retry_interval` as an array of seconds (e.g., `[30, 300, 900]`) to implement progressive backoff. Each retry attempt uses the next interval in the sequence.
  </Accordion>
  <Accordion title="Audience segmentation">
    Target specific users with **Static** audiences (manually curated), **Dynamic** audiences (criteria-based resolution at query time), or **Segments** (reusable cohorts based on profile attributes).
  </Accordion>
  <Accordion title="Open and action tracking">
    Every delivery gets unique open and action slugs. Tracking routes are registered automatically. Optionally integrate with Google Analytics GA4 by setting your `GOOGLE_ANALYTICS_ID`.
  </Accordion>
  <Accordion title="Delivery scheduling">
    Control exactly when notifications reach users — configure allowed days of the week and a time window (hours) per strategy. The scheduler re-dispatches strategies hourly.
  </Accordion>
  <Accordion title="Nova dashboard">
    Manage notifications, templates, audiences, profiles, deliveries, and events directly inside Laravel Nova. No separate admin interface needed.
  </Accordion>
</AccordionGroup>

## Supported channels

| Channel | Identifier | Provider |
|---------|-----------|---------|
| Nova bell | `nova` | Laravel Nova native |
| Card | `card` | Notifications Dashboard |
| Email | `email` | Laravel Mail |
| SMS | `sms` | Twilio |
| Voice call | `call` | Twilio |
| WhatsApp | `whatsapp` | Twilio Content Templates |
| Web Push | `webpush` | Service Workers (HTTPS required) |
| Slack | `slack` | Laravel Slack channel |
| Microsoft Teams | `teams` | laravel-notification-channels/microsoft-teams |

## Requirements

- PHP 8.2+
- Laravel Nova 5.4+
- Laravel application served over HTTPS (required for Web Push)

<Note>
  Notification Center is available on [Packagist](https://packagist.org/packages/opscale-co/notification-center). Install it with `composer require opscale-co/notification-center`.
</Note>
